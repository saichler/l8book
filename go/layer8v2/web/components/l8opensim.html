Network Simulator

Scalable simulator for 25,000+ devices across 28 types.

View on GitHub All Components

Overview

OpenSim is a scalable network and infrastructure simulator. It provides realistic SNMP v2c/v3, SSH, and HTTPS REST API interfaces for testing network management applications, monitoring systems, and automation tools. It can simulate 25,000+ devices with dedicated IP addresses using TUN interfaces and Linux network namespaces.

Protocols

SNMP v2c/v3
MD5/SHA1 authentication, DES/AES128 privacy, OID walks and gets
SSH
VT100 terminal emulation with vendor-specific command sets
HTTPS REST
Storage device simulation with shared TLS certificates

Device Categories

Core Routers (5 types)

Cisco ASR9K (48 ports), Cisco CRS-X (144 ports), Huawei NE8000 (96 ports), Nokia 7750 SR-12 (72 ports), Juniper MX960 (96 ports).

Edge Routers (3 types)

Juniper MX240 (24 ports), NEC IX3315 (48 ports), Cisco IOS (4 ports).

Data Center Switches (2 types)

Cisco Nexus 9500 (48 ports), Arista 7280R3 (32 ports).

Campus Switches (3 types)

Cisco Catalyst 9500, Extreme VSP4450, D-Link DGS-3630.

Firewalls (4 types)

Palo Alto PA-3220, Fortinet FortiGate-600E, SonicWall NSa 6700, Check Point 15600.

Servers (4 types)

Dell PowerEdge R750, HPE ProLiant DL380, IBM Power S922, Linux Server (Ubuntu 24.04 with 36+ SSH commands).

GPU Servers (3 types)

NVIDIA DGX-A100 (8x80GB), DGX-H100 (8x80GB), HGX-H200 (8x141GB). Per-GPU metrics: utilization, VRAM, temperature, power, fan speed, clock speeds.

Storage Systems (4 types)

AWS S3, Pure Storage FlashArray, NetApp ONTAP, Dell EMC Unity. All support SNMP, SSH, and HTTPS REST APIs.

Features

Network Namespace Isolation
Each device runs in its own Linux network namespace
TUN/TAP Integration
Dedicated IP per device via TUN interfaces
Dynamic Metrics
Realistic CPU, memory, temperature, GPU metrics with sine-wave cycling
Entity MIB Alignment
Properly aligned ifTable and Entity MIB data across all devices
CDP and LLDP
Network topology discovery support
OSPF, BGP, VRF
Routing protocol simulation via SSH
Round-Robin Creation
Bulk device creation by category
CSV Export
Device list export and routing script generation

Performance

Tested at 25,000+ concurrent devices
~50MB base + ~1KB per device memory footprint
Pre-generated metrics (100-point sine-wave curves)
Lock-free atomic indexing
Shared SSH and TLS keys across devices
SNMP buffer pooling
Worker-pool pre-allocation for bulk creation (10+ devices)

Deployment

Run directly with root privileges for TUN interface creation. Docker image available as saichler/opensim-web. Kubernetes deployment via StatefulSet in the opensim namespace with hostNetwork: true.
