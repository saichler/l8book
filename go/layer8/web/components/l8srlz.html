Overview

L8SRLZ is a cross-platform binary serialization library for microservices. It handles Go and Java, integrates with Protocol Buffers, supports query-based selective serialization, and includes replica tracking for distributed systems.

Serialization Format

Each value is prefixed with its reflect.Kind as an int32 (4 bytes), enabling type identification during deserialization without prior schema knowledge. Buffers start at 1,024 bytes and double as needed to minimize allocations.

Supported Types

Primitives

Integers: int, int32, int64, uint32, uint64
Floating point: float32, float64
Text: string with length-prefixed encoding
Boolean: bool
Byte: single byte (uint8)

Complex Types

Structs: Protocol Buffers message serialization
Collections: Slices and arrays with type preservation
Maps: Key-value collections with type inference
Pointers: Automatic dereferencing with null safety
